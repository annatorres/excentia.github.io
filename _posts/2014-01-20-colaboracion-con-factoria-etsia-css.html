--- layout: post title: 'Colaboración con Factoría ETSIA: CSS Validator' date: '2014-01-20T07:00:00.000+01:00' author: Antonio Calero categories: - noticias old-tags: - sonarqube - plugin - calidad del software modified_time: '2014-01-20T07:00:03.445+01:00' blogger_id: tag:blogger.com,1999:blog-2090413186558456591.post-7086114749210861230 blogger_orig_url: http://noticias.excentia.es/2014/01/colaboracion-con-factoria-etsia-css.html --- Factoría ETSIA en colaboración con excentia ha desarrollado CSS Validator que analiza recursos CSS localizando fallos de implementación, malas prácticas e incumplimiento de la normativa WCAG 1.0 y WCAG 2.0 para ayudar a la CSS que sea óptima y accesible.

Las características de este nuevo producto son:

  • Análisis a través de SonarQube de gramáticas CSS validando el correcto uso de Keywords, Propiedades y valores introducidos.
  • Más de 30 reglas de buenas prácticas relacionadas con la compresión, adaptabilidad y mal uso de Keywords.
  • Más de 30 reglas adicionales de accesibilidad WEB para ayudar a cumplir al usuario con la normativa WCAG 1.0 y WCAG 2.0. (Regla de contraste de colores, reglas de uso de Keywords necesarios para accesibilidad, reglas de validación de medidas, etc.).
