--- layout: post title: ¡SonarQube 5.0 disponible! date: '2015-01-30T11:15:00.000+01:00' author: Antonio Calero categories: - noticias old-tags: - sonarqube - código abierto - opensource - calidad de software modified_time: '2016-02-11T14:24:30.655+01:00' blogger_id: tag:blogger.com,1999:blog-2090413186558456591.post-3473030183472401254 redirect_from: /2015/01/sonarqube-50-disponible.html --- Estamos muy contentos de poder anunciar que ya está disponible una nueva versión de SonarQube, la 5.0. El equipo de SonarQube ha trabajado mucho para empezar a incluir las mejoras previstas para este año y ya podemos empezar a disfrutarlas :-)

Esta versión incluye las siguientes características:
  • Una página completamente nueva para la gestión de las evidencias, diseñada para trabajar de forma mucho más eficiencia en el día día gracias a:
    • Diferentes facetas que permiten visualizar y navegar desde muchos puntos de vista diferentes
    • Facilidad y rapidez para la navegación entre evidencias y el código fuente
    • Atajos de teclado en las acciones sobre las evidencias (pulsa '?' para descubrirlos)
  • La información del SCM (Source Control Management) ahora esta embebida en el núcleo de la plataforma, con integración nativa para Git y Subversion
    • El resto de proveedores de SCM se incluirán y soportarán a través de plugins
  • Soporte para el caracter '/' en los nombres de las ramas
  • Fin de soporte para Maven 2
Además, hay que tener en cuenta que a partir de ahora Java 7 pasa a ser el entorno de ejecución del servidor.

Si necesitas saber más, consulta las notas de versión directamente en el JIRA de SonarQube y no dudes en consultar todo lo que tienes que tener en cuenta para la actualización.
