Desde el viernes está disponible una nueva versión (0.2) de la extensión de Sonar para Apache JMeter que permite incluir las métricas de las pruebas de rendimiento en tu cuadro de mandos de Sonar. 


Esta nueva versión incluye nuevas características que hacen más fácil su utilización, y que permite integrar el ciclo de pruebas de rendimiento en Maven de forma automatizada: 
  • Se ha verificado el funcionamiento con la nueva versión 2.5 de Apache JMeter.
  • Se incluye una opción para poder usar el plugin sin necesidad del componente del servidor, para ello basta con establecer una ruta local al fichero de resultados de JMeter desde tu configuración del proyecto en Sonar.
  • Se ha añadido una opción para establecer la unidad de tiempo (segundos, minutos, horas). - Se incluye la posibilidad de utilizar el plugin junto a otras extensiones, como Maven JMeter Plugin y Cargo, de manera que se pueda ejecutar en una sola fase de Maven, las pruebas automatizadas, y la captura de resultados. 
La nueva versión está disponible para su descarga a través del centro de actualizaciones de Sonar.
