excentia lanza la primera versión pública (1.1.1) de Sonar Findbugs Model Plugin, una nueva extensión de Sonar que permite seguir ampliando la herramienta con nuevos modelos para poder evaluar nuevas características de calidad de los proyectos de desarrollo de software. 

De forma similar al modelo de calidad de PMD, este plugin o extensión permite implementar el modelo de calidad de la herramienta Findbugs para poder clasificar las reglas en sus diferentes categorías (rendimiento, malas prácticas, seguridad, internacionalización, etc.).

Sonar Findbugs Model Plugin añade esa clasificación dentro de Sonar y da "semántica" a las reglas para poder atacar problemas específicos de calidad en un proyecto.

En muchas ocasiones a la hora de resolver evidencias en Sonar no sabes exactamente que mejoras vas a obtener... Con esta extensión podrás decidir que aspectos vas a mejorar gracias a la clasificación en las diferentes categorías (agrupaciones de reglas).
