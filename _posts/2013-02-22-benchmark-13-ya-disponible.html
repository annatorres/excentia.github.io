Ya está disponible la nueva versión de Sonar Benchmark Plugin para realizar benchmarking de la calidad de tus proyectos de software.

La nueva versión incluye:

  • Nuevo mecanismo de seguridad activado por defecto para que las comparativas se realicen solo con los proyectos que el usuario conectado puede ver. No obstante se puede desactivar para mantener el comportamiento anterior.
  • Nueva métrica para comparativas de desarrolladores: ahora se muestra el índice de cumplimiento de reglas por desarrollador.
  • Se ha mejorado la visualización del widget de comparativa de los desarrolladores.
¡Descarga y prueba el plugin durante dos semanas de forma gratuita y cuéntanos que te parece!
